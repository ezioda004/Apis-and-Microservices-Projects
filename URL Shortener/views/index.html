URL Shortener

User stories:

  1. Users can make a POST request to http://localhost:1337/api/shorturl/new and will receive a shortened link in response.
  2. If the link is invalid, the user gets response {"error":"invalid URL"}
  3. When the user visits the shortened link, they are redirected to the original link.

Short URL Creation

Example: POST localhost:1337/api/shorturl/new - https://www.google.com

Example Usage:

localhost:1337/api/shorturl/0

Will Redirect to:

https://www.google.com
